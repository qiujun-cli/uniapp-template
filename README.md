## uni-cli-template

#### 技术栈

- Node 16.20.1
- pnpm 8.6.3
- vue 3.4
- typescript
- pinia
- uniapp

#### 使用

```bash
npm i @qiu_jun/qjcli

qjcli create <projectname> #选择uni-template

# 如果没有pnpm 项目使用pnpm，commit自动校验都是用pnpm,请使用pnpm启动项目
npm install pnpm -g

pnpm install

pnpm dev:weapp
```
